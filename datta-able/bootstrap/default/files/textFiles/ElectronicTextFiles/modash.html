מערכות אופטיות משמשות את לוחם מודיעין השדה לתצפית והכוונת אש מדוייקת ביום ובלילה. המערכות האופטיות כוללות מכשור מדויק מהמשפחות הבאות: כוונות מסוגים שונים, משקפות, מצפנים, זרקורים, סמני לייזר, מכשירי ראיית לילה, אמצעי תצפית ועוד.


תפקידים במערך הנפרס
  • תפעול, בדיקה ואיתור תקלות באמצעים האופטיים השונים, מערכות תצפית ושו"ב.

  • תיקון תקלות וכיולים של אמצעים אופטיים.

  • ביצוע בדיקה חודשית לכלל האמצעים האופטיים ביחידה, כולל בדיקות ויזואליות (מכניות), ניקיון אופטי ( עפ"י הוראות טכניות) ותפעול של האמצעים בהתאים להוראות הבטיחות.

  • אחזקת שבר: איתור ותיקון תקלות בדרג א' באמצעים האופטיים השונים והחזרה לכשירות על פי קריאה מזדמנת מהלוחמים ובהתאם לסימפטומים תפעוליים.
