המכללה לחדשנות ומנהיגות טכנו-לוגיסטית

הכירו את מכללת "לוגי-טק"

המרכז הטכנו-לוגיסטי למקצועות המחר, מכללה למנהיגות, פיקוד והכשרה בעידן היבשה החדשה בבאר יעקב

אשר יהווה מודל ראשוני ופורץ דרך לחדשנות בתחומי ה"למידה המשמעותית" הנדרשת למקצועות המחר במציאות טכנולוגית משתנה במגמות רבות וייחודיות כגון: הנדסת אלקטרוניקה ומחשבים, מכטרוניקה ותעשייה וניהול. המרכז יכשיר, יסמיך ויעצים מלש"בים, חיילים ומפקדים וישמש אבן שואבת לכ"א ייחודי ואיכותי בעלי הזדהות גבוהה וגאוות יחידה, אשר יהוו את שדרת הפיקוד העתידית בצבא הגנה לישראל. מכללת לוגיטק תקנה לכל תלמיד תנאים וסל הטבות ייחודיות ונרחבות.
